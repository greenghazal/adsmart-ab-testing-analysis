# AdSmart A/B Testing Analysis

Statistical analysis of an A/B testing campaign evaluating the effectiveness of AdSmart digital advertisements on user conversion rates.

## Key Findings
- **Conversion Rate Impact**: Ad group showed significantly higher conversion (2.55%) vs PSA control (1.78%)
- **Statistical Significance**: Z-test p-value < 0.001 - ad had real positive impact
- **Optimal Timing**: Monday best day, 4 PM most effective hour
- **Ad Exposure Correlation**: Converters saw significantly more ads than non-converters (t-test, p < 0.001)
